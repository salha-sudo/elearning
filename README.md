#  Plateforme E-Learning

## Description du projet
Ce projet est une plateforme d’apprentissage en ligne développée avec Spring Boot.  
Elle permet aux utilisateurs (apprenants et formateurs) de gérer des cours, inscriptions et progressions.


##  Sprint 1 : Initialisation du projet
- Création du projet Spring Boot
- Configuration de la base de données MySQL
- Mise en place des entités principales :
  - Utilisateur
  - Cours
- Mise en place des repositories JPA
- Connexion à la base de données avec XAMPP



##  Sprint 2 : Gestion des cours
- Création du module Cours
- Ajout / modification / suppression des cours
- Affichage du catalogue des cours
- Interface formateur (CRUD des cours)
- Intégration Thymeleaf pour les vues



## Sprint 3 : Module apprenant
- Inscription des apprenants aux cours
- Gestion des inscriptions
- Suivi de progression
- Validation des leçons complétées
- Affichage du tableau de progression



##  Technologies utilisées
- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- Thymeleaf
- MySQL (XAMPP)
- Maven
